Pivot Table - Save & Load Layouts

Customize your pivot table and save different views for quick access

The Pivot Table Sales by Item page now supports saving, loading, and managing custom layouts. Configure the pivot table the way you need it, save that configuration with a name, and quickly switch between different views.

Toolbar Buttons

New
Resets the pivot table back to the default layout
Save
Saves changes to the currently loaded layout
Save As
Saves the current layout with a new name
Rename
Renames the currently loaded layout
Remove
Deletes the currently loaded layout
Load
Opens a dropdown to select a previously saved layout

How To Use

Save a New Layout

  1. Configure the pivot table the way you want it (drag fields, expand/collapse rows, adjust filters, etc.)
  2. Click Save As in the toolbar
  3. Enter a name for your layout (e.g., "By Quarter - Top Customers")
  4. Click OK

Load a Saved Layout

  1. Click the Load dropdown in the toolbar
  2. Select the layout name from the list
  3. The pivot table will update to match that saved configuration

Update an Existing Layout

  1. Load the layout you want to update
  2. Make your changes to the pivot table
  3. Click Save to overwrite the layout with your current configuration

Rename a Layout

  1. Load the layout you want to rename
  2. Click Rename in the toolbar
  3. Enter the new name and click OK

Delete a Layout

  1. Load the layout you want to delete
  2. Click Remove in the toolbar
  3. Confirm the deletion

Reset to Default

  1. Click New in the toolbar to return to the original default pivot table configuration
  2. If you want to keep this as a named layout, use Save As

Good to Know

  • Layouts are saved per user, so each rep has their own set of saved layouts
  • Layouts are stored on the server and available from any device or browser you log in from
  • The Field List and Export buttons continue to work as before
  • You can create as many saved layouts as you need
